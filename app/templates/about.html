About

M1CR0B1AL1Z3R allows generating both quantitative and qualitative measurements for ad-hoc downstream analyses given a microbial genomic sequences dataset. Visual and textual results, ready for publication or further analysis, are given as an output. For example, open reading frames of each genome, orthologous groups (both unaligned and aligned), core proteome, phylogenetic tree, GC content and more. See running example input and output.
